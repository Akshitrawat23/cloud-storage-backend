# ☁️ Cloud File Storage System

A secure cloud-based file storage backend built with Flask, MongoDB, JWT Authentication, and AWS S3.

## Features

- User Registration
- User Login
- Password Hashing
- JWT Authentication
- MongoDB Integration
- REST APIs

## Tech Stack

- Python
- Flask
- MongoDB
- JWT
- AWS S3 (In Progress)
- Docker (Planned)

## API Endpoints

POST /register
POST /login
